# ATM Machine Operations System

A simple console-based ATM banking application developed using Java.

## Features

- User Login Authentication
- Saving and Current Account Selection
- Balance Inquiry
- Deposit Money
- Withdraw Money
- HashMap-based Account Storage

## Technologies Used

- Java
- HashMap
- Scanner Class

## How to Run

Compile:

```bash
javac ATM.java
```

Run:

```bash
java ATM
```
